# Zoomcab Frontend 🛺

A premium, "Cyberpunk India" styled ride-booking interface.

## Features
-   **Dark Glassmorphism UI**: Inspired by "Cyberdrive".
-   **Interactive Map**: Dark mode Leaflet map centred on Bangalore.
-   **3D Elements**: Interactive Payment Card.
-   **Animations**: Smooth page transitions and radar scanning effects.

## How to Run

1.  Navigate to the directory:
    ```bash
    cd c:\Users\Guru\Desktop\Projects\DS\Zoomcab
    ```

2.  Install dependencies (if not already done):
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:5173](http://localhost:5173) in your browser.

## Project Structure
-   `src/pages/Login.jsx`: Entry point with animated form.
-   `src/pages/Home.jsx`: Main booking interface with Map.
-   `src/pages/Payment.jsx`: Payment gateway simulation.
-   `src/index.css`: Global "Cyber" theme and animations.
